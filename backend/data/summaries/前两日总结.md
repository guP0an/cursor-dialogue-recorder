## 📌 如何用本网站「总结前两天」

1. **确保有对话数据**  
   前两天（1/27、1/28）要在 Cursor 中有对话，并已通过 Cursor Hooks 记录到本网站（后端运行且 Hook 已配置）。

2. **用网站生成某日总结**  
   - 方法一：访问前端，若已支持「按日期分析」则选择 1/27、1/28 触发分析。  
   - 方法二：直接调接口  
     ```bash
     curl -X POST http://localhost:3001/api/analyze/2026-01-27
     curl -X POST http://localhost:3001/api/analyze/2026-01-28
     ```  
   有对话的日期会生成 `backend/data/summaries/YYYY-MM-DD.md`。

3. **查看总结**  
   - 前端：打开「每日总结」或对应日期的总结页。  
   - 本地文件：`backend/data/summaries/` 下对应日期的 `.md` 文件。
