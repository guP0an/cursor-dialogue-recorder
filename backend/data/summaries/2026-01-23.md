## 💡 知识点总结

### 1. React 性能优化方法

**核心知识点：**
- **React.memo**: 防止不必要的重新渲染，通过记忆化组件来优化性能
- **useMemo 和 useCallback**: 缓存计算结果和函数引用，避免重复计算
- **代码分割**: 使用 `React.lazy` 和 `Suspense` 实现按需加载，减少初始包大小
- **虚拟化长列表**: 使用 `react-window` 或 `react-virtualized` 处理大量数据渲染
- **优化状态管理**: 避免不必要的状态更新，合理组织状态结构

**应用场景：**
- 大型React应用性能优化
- 处理大量数据渲染的场景
- 减少不必要的重渲染

### 2. TypeScript 泛型（Generics）

**核心概念：**
TypeScript中的泛型是一种创建可重用组件的工具，允许创建可以处理多种数据类型的组件，而不是单一的数据类型。

**基本语法：**
```typescript
function identity<T>(arg: T): T {
  return arg;
}
```

**关键点：**
- `<T>` 是类型变量，可以在函数中使用
- 提供类型安全的同时保持代码的灵活性
- 适用于函数、接口、类等多种场景

**应用价值：**
- 提高代码复用性
- 保持类型安全
- 减少重复代码

### 3. React 组件创建方法

**推荐方式：函数式组件**

```jsx
function MyComponent() {
  return <div>Hello World</div>;
}
```

**或使用箭头函数：**

```jsx
const MyComponent = () => {
  return <div>Hello World</div>;
};
```

**要点：**
- 函数式组件是现代React开发的标准方式
- 简洁、易读、易测试
- 支持Hooks，功能强大
