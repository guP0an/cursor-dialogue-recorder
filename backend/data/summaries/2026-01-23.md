## 💡 知识点总结

优化React应用性能的几种方法：

1. **使用React.memo**: 防止不必要的重新渲染
2. **使用useMemo和useCallback**: 缓存计算结果和函数
3. **代码分割**: 使用React.lazy和Suspense
4. **虚拟化长列表**: 使用react-window或react-virtualized
5. **优化状态管理**: 避免不必要的状态更新

TypeScript中的泛型（Generics）是一种创建可重用组件的工具，它允许你创建可以处理多种数据类型的组件，而不是单一的数据类型。

基本语法：
```typescript
function identity<T>(arg: T): T {
  return arg;
}
```

这里的 `<T>` 是类型变量，可以在函数中使用。

要创建一个React组件，你可以使用函数式组件或类组件。函数式组件是推荐的方式：

```jsx
function MyComponent() {
  return <div>Hello World</div>;
}
```

或者使用箭头函数：

```jsx
const MyComponent = () => {
  return <div>Hello World</div>;
};
```
